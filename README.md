# Barbershop Booking Bot

Telegram бот для записи клиентов в барбершоп. Позволяет выбрать специалиста, дату и время для посещения.

## Возможности

- Выбор конкретного специалиста
- Выбор услуги без привязки к специалисту
- Просмотр доступных дат для записи
- Выбор свободного времени
- Автоматическое удаление занятого слота из базы данных

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/barbershop-bot.git
cd barbershop-bot
```

2. Установите зависимости:
```bash
pip install pytelegrambotapi
```

3. Получите токен бота у [@BotFather](https://t.me/botfather) в Telegram

4. Замените `'Ваш токен'` в коде на полученный токен:
```python
bot = telebot.TeleBot('Ваш токен')
```

## Настройка базы данных

Бот использует SQLite для хранения расписания. Для первоначальной настройки:

Заполните таблицу `barbershop` доступными слотами:
   - `barber_name` - имя специалиста
   - `app_date` - дата в формате 'YYYY-MM-DD'
   - `app_time` - время в формате 'HH:MM'

Пример добавления данных:
```sql
INSERT INTO barbershop (barber_name, app_date, app_time) VALUES 
('Антон', '2024-01-20', '10:00'),
('Антон', '2024-01-20', '11:00'),
('Вячеслав', '2024-01-20', '10:00');
```

## Использование

1. Запустите бота:
```bash
python bot.py
```

2. В Telegram найдите своего бота и отправьте команду `/start`

3. Следуйте инструкциям бота:
   - Выберите опцию записи
   - Выберите специалиста или услугу
   - Выберите дату
   - Выберите время
